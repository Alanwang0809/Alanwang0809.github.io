## 🎯 提醒需求分析

### 原始需求
```
每月9日下午5点提醒
↓
如果9日是周末（周六或周日）
↓
则顺延到下周一下午5点
↓
提醒内容：问店里要水电费、推广费和物业费的账单
↓
询问对象：店长或者物业
```

### 复杂点分析
1. **日期判断**：需要判断每月9日是否为周末
2. **顺延逻辑**：周末顺延到下周一下午5点
3. **月度循环**：每月都需要执行
4. **时间准确**：下午5点准时提醒

---

## ⚙️ 技术实现方案

### 方案选择：双提醒系统
由于cron表达式无法直接进行日期条件判断，采用**双提醒互补系统**：

#### 提醒1：每月9日固定提醒
- **时间**：每月9日下午5点
- **cron表达式**：`0 17 9 * *`
- **逻辑**：
  - 如果9日是工作日 → 正常提醒
  - 如果9日是周末 → 提醒中说明"如果是周末请忽略"

#### 提醒2：每周一补充提醒
- **时间**：每周一下午5点
- **cron表达式**：`0 17 * * 1`
- **逻辑**：
  - 如果上周9日是周末 → 这次是顺延提醒
  - 如果上周9日是工作日 → 这次是普通周一提醒

### 智能判断流程
```
用户收到提醒 → 自行判断：
1. 如果是每月9日（且是工作日）→ 立即处理
2. 如果是每月9日（但是周末）→ 忽略，等待下周一
3. 如果是周一（且上月9日是周末）→ 这是顺延提醒，立即处理
4. 如果是周一（但上月9日是工作日）→ 这是普通周一提醒，忽略账单事项
```

## 📝 账单详情说明

### 需要索要的账单类型
1. **水电费账单**
   - 电费：用电量、单价、总金额
   - 水费：用水量、单价、总金额
   - 可能还有燃气费等

2. **推广费账单**
   - 商场/物业的推广活动费用
   - 广告位费用
   - 促销活动分摊费用

3. **物业费账单**
   - 物业管理费
   - 公共区域维护费
   - 保洁、保安等服务费用
   - 其他杂费

### 索要渠道
1. **首选**：店长
   - 店长可能有账单副本
   - 店长了解店铺运营情况
   - 店长可以协助沟通

2. **备选**：物业办公室
   - 直接获取官方账单
   - 确认费用明细
   - 解决疑问和争议

3. **其他渠道**：
   - 商场管理处
   - 财务部门
   - 线上缴费系统

### 处理流程建议
```
1. 提前准备：准备好店铺信息、合同编号等
2. 联系对方：电话或当面联系店长/物业
3. 明确需求：说明需要水电费、推广费、物业费账单
4. 获取方式：询问是电子版还是纸质版，如何获取
5. 确认时间：确认账单周期和缴费截止日期
6. 记录信息：记录联系人、获取方式、预计时间
7. 跟进确认：确保按时收到账单
```

## ⏰ 时间管理建议

### 最佳处理时间
- **提醒时间**：下午5点
- **最佳联系时间**：工作日下午2-4点
- **避免时间**：午休时间（12:00-13:30）、下班前后

### 处理时限建议
```
提醒日（9日或顺延周一）：
- 当天：联系店长或物业，提出需求
- 1-2天内：获取账单电子版或查看方式
- 3-5天内：完成账单核对和问题解决
- 7天内：完成缴费或安排付款

关键节点：
- 联系：提醒当天完成
- 获取：2天内完成
- 核对：5天内完成
- 处理：7天内完成
```

### 逾期风险
1. **滞纳金**：逾期缴费可能产生额外费用
2. **服务影响**：严重逾期可能影响水电供应
3. **信用影响**：影响店铺在商场的信用记录
4. **法律风险**：长期拖欠可能面临法律诉讼
