# 🦞 小龙虾交易员系统

## 系统架构

### 核心模块
1. **数据层** - 市场数据获取和存储
2. **分析层** - 技术分析和基本面分析
3. **策略层** - 交易策略生成
4. **执行层** - 交易执行和风控
5. **监控层** - 实时监控和警报

### 支持市场
- A股（沪深）
- 港股
- 美股
- 加密货币
- 外汇
- 期货（计划中）

## 技术栈
- Python 3.8+
- yfinance / pandas-datareader
- pandas-ta / TA-Lib
- scikit-learn / TensorFlow（机器学习）
- FastAPI（API服务）
- Redis（缓存）
- PostgreSQL（数据存储）

## 部署架构
```
用户界面（飞书/Web） → API网关 → 交易引擎 → 市场数据源
                                    ↓
                                风控系统
                                    ↓
                                执行系统 → 券商API
```
