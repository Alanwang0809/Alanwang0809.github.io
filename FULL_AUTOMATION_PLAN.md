## 🔐 GitHub自动化部署配置

### 步骤1：生成GitHub PAT
1. 访问: https://github.com/settings/tokens
2. 点击 "Generate new token"
3. 选择 "Fine-grained tokens"
4. 权限设置：
   - Repository access: All repositories 或 Only select repositories
   - Permissions:
     - Contents: Read and write
     - Pages: Read and write
     - Metadata: Read-only
5. 生成token并保存

### 步骤2：配置自动化脚本
将PAT添加到自动化脚本中，实现：
- 自动创建GitHub仓库
- 自动推送代码
- 自动启用GitHub Pages

### 步骤3：测试自动化流程
测试一次完整流程，确保一切正常。
