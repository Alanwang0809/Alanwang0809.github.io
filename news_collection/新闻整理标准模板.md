# 📰 新闻整理标准模板

## 🎯 标准要求（Alan要求）

每一条新闻必须包含以下五个要素：

### 1. 时间
- **格式**：YYYY-MM-DD
- **要求**：新闻发生或发布时间
- **示例**：2026年3月16日

### 2. 主体
- **格式**：涉及的国家、公司、人物、机构等
- **要求**：清晰明确，多个主体用顿号分隔
- **示例**：中国商务部、美国贸易代表办公室

### 3. 事件（2026-03-16优化要求：更加详细）
- **格式**：具体发生了什么，包含背景、细节、各方反应
- **要求**：描述准确、完整、详细，包含：
  1. 事件发生的具体背景和原因
  2. 涉及各方的具体行动和表态
  3. 事件的具体过程和细节
  4. 相关数据和事实支撑
  5. 各方的反应和后续计划
- **示例**：美国贸易代表办公室宣布对包括中国在内的60个经济体发起新一轮301调查，指控这些经济体存在"不公平贸易行为"。中国商务部新闻发言人在当天下午的例行记者会上回应称，美方此举"严重扰乱国际经贸秩序"，是典型的单边主义和保护主义行为。商务部表示，中国将采取必要措施维护自身合法权益，并呼吁美方回到多边贸易体制的正确轨道上来。

### 4. 影响（2026-03-16优化要求：更加详细）
- **格式**：要点列表（至少5点），包含量化分析和多维度影响
- **要求**：分析深入、详细、全面，包含：
  1. 从新闻中摘取的具体影响
  2. 专业的判断和预测
  3. 量化数据和指标支撑
  4. 多个维度的影响分析（经济、政治、社会、技术等）
  5. 短期、中期、长期的影响预测
- **示例**：
  - **贸易摩擦升级**：可能引发新一轮中美贸易摩擦，影响双边贸易额预计超过2000亿美元
  - **供应链冲击**：全球供应链将面临新的不确定性，特别是高科技和制造业供应链
  - **企业成本上升**：相关出口企业可能面临关税增加，生产成本预计上升10-15%
  - **投资环境变化**：跨国企业投资决策将更加谨慎，可能影响中国吸引外资
  - **多边体系受损**：WTO多边贸易体系权威性进一步受损，全球贸易规则面临挑战

### 5. 原文链接
- **格式**：完整的HTTPS链接
- **要求**：可点击验证，来源可靠
- **示例**：https://finance.sina.com.cn/jjxw/2026-03-16/doc-inhrcpie9922556.shtml

## 🏗️ 新闻分类标准

### 一级分类
1. **国际新闻**：涉及国际关系、地缘政治、全球经济等
2. **国内新闻**：中国国内政治、经济、社会等
3. **AI与科技**：人工智能、科技创新、互联网等
4. **新能源**：新能源汽车、电池技术、可再生能源等
5. **商业航天**：卫星、火箭、太空探索等
6. **经济金融**：股市、货币政策、投资等
7. **社会民生**：教育、医疗、消费等

### 二级分类（可选）
- 按地区：亚洲、欧洲、美洲等
- 按行业：半导体、汽车、金融等
- 按重要性：重大、重要、一般

## 🔧 整理流程

### 第一步：新闻收集
1. 使用web_fetch工具获取新闻网页
2. 提取关键信息：标题、时间、主体、事件
3. 保存原文链接

### 第二步：信息整理
1. 按照标准格式整理每个要素
2. 分析事件影响（至少3点）
3. 验证链接有效性

### 第三步：质量检查
1. 检查五个要素是否齐全
2. 验证信息准确性
3. 确保格式规范

### 第四步：文件保存
1. 使用标准文件名：YYYY-MM-DD_晨间新闻汇总.md
2. 使用UTF-8编码保存
3. 按照分类组织内容

## 📊 质量标准

### 准确性要求
1. **时间准确**：新闻发生时间必须准确
2. **主体明确**：涉及主体必须清晰
3. **事件真实**：事件描述必须基于事实
4. **影响合理**：影响分析必须有逻辑依据
5. **链接有效**：原文链接必须可访问

### 完整性要求（2026-03-16优化）
1. **要素齐全**：五个要素缺一不可
2. **内容完整详细**：事件描述必须详细，包含背景、细节、各方反应
3. **分析深入详细**：影响分析必须详细，包含量化数据、多维度分析、专业判断
4. **格式规范**：严格按照模板格式
5. **数据支撑**：必须有具体数据、事实、指标支撑
6. **多维度覆盖**：覆盖经济、政治、社会、技术等多个维度

### 实用性要求
1. **投行相关**：优先选择与投行工作相关的新闻
2. **投资价值**：关注有投资机会的新闻
3. **趋势洞察**：能够反映行业趋势
4. **决策支持**：为投资决策提供信息支持
