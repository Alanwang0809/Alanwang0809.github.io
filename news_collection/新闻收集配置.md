## 🎯 新闻收集标准

### 新闻重要性标准
```
1. 政治类：国家领导人活动、重大政策发布、国际关系变化
2. 经济类：宏观经济数据、重要经济政策、金融市场波动
3. 科技类：重大技术突破、科技公司动态、AI/新能源进展
4. 金融类：股市债市动态、央行政策、重要企业财报
5. 社会类：重大社会事件、民生政策、环境气候新闻
6. 国际类：国际冲突、多边合作、全球治理议题
```

### 新闻来源优先级
```
第一梯队（权威媒体）：
- 国内：新华社、人民日报、央视新闻、财新网
- 国际：BBC、Reuters、AP、Bloomberg、WSJ

第二梯队（专业媒体）：
- 财经：财联社、第一财经、华尔街见闻
- 科技：36氪、钛媒体、TechCrunch
- 国际：CNN、NYT、FT、The Economist

第三梯队（聚合平台）：
- 今日头条热点、微博热搜、百度热榜
- Reddit热门、Twitter趋势、Google新闻
```

### 筛选标准
```
1. 时效性：过去24小时内发生或报道
2. 重要性：影响范围广，关注度高
3. 相关性：与Alan工作（投行、AI、新能源）相关
4. 可信度：来自权威或可靠信源
5. 多样性：覆盖多个领域和地区
```
